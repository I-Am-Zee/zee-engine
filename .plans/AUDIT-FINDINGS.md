# Audit Findings — April 2026 Forensic Review

This document tracks the architectural and security debt identified during the "Pure Engine" silo migration.

## 🔴 CRITICAL (Immediate Fix Required)
- **[AF-001] Unauthenticated Webhook:** The `/api/checkout/order-completed` endpoint accepts any POST request without validating Snipcart's signature (`X-Snipcart-RequestToken`).
- **[AF-002] Free Product Logic Risk:** `src/config/ecommerce.ts` defaults missing or malformed prices to `0`. Combined with AF-001, this allows for effortless price spoofing.
- **[AF-003] Missing Collection Export:** The `blog` collection is defined in `config.ts` but not exported. `getCollection('blog')` returns undefined, breaking all blog features.

## 🟠 MAJOR (Functional/UX Debt)
- **[AF-004] URL Slug Desync:** The Navbar uses a `slugify()` utility, while the Footer uses a manual regex. Category names with special characters (e.g., "Rings & Sets") will have different URLs, causing 404s.
- **[AF-005] Semantic Main Violation:** All Legal and Brand story pages have a `<main>` tag inside another `<main>` tag from `BaseLayout`. This is a W3C violation and hurts accessibility.
- **[AF-006] SideDrawer State Leak:** The SideDrawer relies on `window.ENGINE_UPSELL_CONTEXT`. This global can be clobbered or become stale if multiple items are added rapidly.
- **[AF-007] Affiliate Gating Failure:** The Footer displays the "Shop" column regardless of `isAffiliate` status, leading to dead-ends on affiliate sites.

## 🟡 MINOR (Design/Cleanup)
- **[AF-008] Orphaned Pages:** FAQ and Contact remain in the root `/pages/` while all other editorial content has been siloed into `/legal/` or `/brand/`.
- **[AF-009] Design Token Leakage:** Mobile wishlist count uses `bg-red-500` instead of `bg-accent-coral`.
- **[AF-010] Redundant Data Fetching:** Newsletter settings are fetched in multiple places instead of being passed as props.
- **[AF-011] Weight Rounding Risk:** `Math.round()` on weights < 0.5kg results in 0kg, which can cause Snipcart shipping calculation failures.
